# KQI-NET

This repository provides a scheme to calculate the KQI of neural network with low memory overhead.

## Installation

#### Install from Github

```bash
git clone https://github.com/Girafboy/KQI-NET.git
cd KQI_NET
```

#### Python Dependencies

```
pytorch
numpy
pytest
```

## Running Tests

To run the test, execute the code:

```bash
python3 test.py
```
